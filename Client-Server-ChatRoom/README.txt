1.) First, Run "ChatServer.java" to connect to port 3000. Next, Run "ChatClient.java" as many times as you want to
connect multiple clients. Then you will have to go back and click on the first client you connected to. 

2.)Enter in a "Username" and the AccessCode "Password" for each of the clients.

3.)Pick a client to send a messege from a chat away!


Notes
-All messeages will display to the server
-To close the program Terminate the Server (If you terminate the Client you will get errors)
